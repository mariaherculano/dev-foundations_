# 🎓 Dev Foundations — Projetos Acadêmicos de Programação

## 📌 Visão Geral
Este repositório reúne uma coleção de projetos acadêmicos e pessoais desenvolvidos durante minha formação em **Análise e Desenvolvimento de Sistemas**.

O objetivo é consolidar conhecimentos práticos em **programação, lógica, desenvolvimento web e manipulação de dados**, utilizando diferentes linguagens e tecnologias.

Os projetos aqui presentes representam exercícios práticos, experimentos e aplicações simples voltadas ao aprendizado de conceitos fundamentais de desenvolvimento de software.

---

## 🎯 Objetivos do Repositório

- Consolidar conhecimentos em **lógica de programação e desenvolvimento de software**
- Praticar **boas práticas de organização de código**
- Desenvolver aplicações simples utilizando diferentes tecnologias
- Demonstrar evolução técnica ao longo da formação acadêmica
- Criar um portfólio de projetos para fins educacionais e profissionais

---

| Categoria | Tecnologia | Aplicação |

| Linguagem | Python | Scripts, CRUD simples e lógica de programação |
| Linguagem | JavaScript | Interatividade e manipulação do DOM |
